# 💧 Automated Water Consumption Monitoring System

An IoT-based smart system that monitors real-time water levels using 
ESP8266 NodeMCU, ultrasonic sensor, relay module, and the Blynk mobile app.

## 🔧 Hardware Components
- ESP8266 NodeMCU (microcontroller)
- Ultrasonic Sensor (HC-SR04)
- Relay Module (controls 230V AC pump)
- LCD Display with I2C Module
- 5x LEDs + 220Ω Resistors
- 230V AC Water Pump

## 📱 Software & Platform
- Arduino IDE
- Blynk App (mobile dashboard)
- Wi-Fi cloud data transmission

## ⚙️ How It Works
1. Ultrasonic sensor detects water level in tank
2. ESP8266 processes and sends data to Blynk Cloud via Wi-Fi
3. User views real-time level on Blynk mobile dashboard
4. Relay automatically controls water pump
5. Push alerts sent for overflow or low water level

## 📊 Key Features
- Real-time water level monitoring
- Automated pump ON/OFF control
- Mobile push notifications via Blynk
- LED indicators for 5 water levels
- Remote monitoring from anywhere
